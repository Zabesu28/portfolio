AP Digicode

AP en C# où je devais créer une interface de connexion ainsi qu'une interface de cryptage de mot de passe. Ces mots de passe devait être enregistré dans un fichier csv. J'ai été chef de projet.

AP GSB Vistat

AP où l'objectif était d'exporter des données de rêquetes SQL au format csv et de les représenter sous forme de graphique dans excel.

AP Centrecall Tickets

AP en C# où on avait un début de projet de gestion de tickets sur lequel on devait coder de nouvelles fonctionnalités, avec des fichiers csv en guise de base de données.

AP Repas

Pour cette AP, on avait un site de gestion de repas en PHP de fournit et on avait pour objectif de rajouter des fonctionnalités, tout en respectant la méthode MVC.

AP Symfony

AP où il fallait faire un site pour un salon de coiffure en utilisant le framework Symfony. Nous avons avons aussi utilisé GitKraken pour rendre plus efficace le travail d'équipe.

Gaming Simulator

Projet dont l'objectif était d'expérimenter la méthode agile scrum. Nous avons du créer un site assez simple en php sur le thème des jeux vidéos. J'ai été chef de projet.

AP Référencement

Une AP qui consistait à s'informer sur le référencement et réaliser une doc sur celui-ci.

AP GSB-gesAMM

AP où l'objectif était de créer une application de gestion de médicaments avant leur mise sur le marché. Cette application a été codé en C# et nous devions la faire communiquer avec SQL Server pour récupérer et traiter les données.

AP Cerza

AP où l'objectif était de créer un site interne pour le personnel du Zoo de Cerza en React, permettant de faire gagner du temps dans les activités de toutes l'équipe du Zoo. Une api en node js a été utilisé. J'ai été chef de projet.
